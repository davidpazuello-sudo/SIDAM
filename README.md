# 🏛️ SIDAM - Sistema de Dívida Ativa Municipal
### **Plataforma de Inteligência Fiscal e Jurídica (MetaGov Framework)**

O SIDAM é um ecossistema de alta tecnologia para gestão, arrecadação e recuperação de créditos da Fazenda Pública.

## 🚀 Stack Única (Restrições Absolutas)
- **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Shadcn/UI.
- **Backend:** Supabase (PostgreSQL + PostGIS).
- **IA:** Google Gemini 1.5 Pro (via AI Studio & Edge Functions).
- **Arquitetura:** UI Engine Dinâmica (Configuração via banco, sem CRUD manual).

## 🗺️ Fases do Projeto
1. **Fase 1-2:** Inscrição, Arrecadação PIX e Negociação Ativa.
2. **Fase 3-4:** Judicialização (TJAM), Contabilidade (AFIM) e Portal Gov.br.
3. **Fase 5-6:** Inteligência (Rating A-D, JUCEA) e Jurídico 4.0 (Minutas IA).
4. **Fase 7-8:** Geoprocessamento, Blockchain Fiscal e Securitização.

## 🛡️ Camadas de Proteção (Voz do Diabo)
- **Outbox Pattern:** Resiliência em pagamentos e baixas.
- **Blockchain Ledger:** Imutabilidade via encadeamento de Hashes SHA-256.
- **Circuit Breaker:** Proteção contra quedas de APIs externas.
- **GIGO Filter:** Estágio de quarentena para dados de secretarias.
