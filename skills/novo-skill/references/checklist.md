# Checklist de validação do skill

- [ ] Nome do skill em lowercase com hífens.
- [ ] Frontmatter com apenas `name` e `description`.
- [ ] Descrição inclui claramente gatilhos de uso.
- [ ] Instruções no corpo são objetivas e acionáveis.
- [ ] `agents/openai.yaml` está alinhado ao propósito do skill.
- [ ] Recursos opcionais (`scripts`, `references`, `assets`) só existem se forem úteis.
- [ ] Validação executada com `quick_validate.py` sem erros.
